## ◉ DocSearch — Hybrid Multilingual Retrieval Engine

> Offline semantic document retrieval built for messy real-world data.

```txt
OCR → Semantic Chunking → Hybrid Retrieval → Reranking
```

### Architecture
- TF-IDF + FAISS hybrid retrieval
- multilingual embeddings
- OCR pipeline for scanned documents
- semantic chunking
- reranking pipeline
- local/offline document search

### Stack
```bash
Python • FAISS • PyTorch • OCR • NLP • Information Retrieval
```

### Why It Matters
Traditional search breaks on multilingual and noisy documents.  
DocSearch applies production retrieval concepts to local-first intelligent search systems.
